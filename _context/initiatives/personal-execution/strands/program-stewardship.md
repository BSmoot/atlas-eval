# Work Strand: Program Stewardship

**ID:** WS-003
**Status:** Active
**Created:** 2026-01-26
**Owner:** Ben Smoot

---

## Purpose

Ensure intended outcomes are achieved throughout strategic programs (Programs 01-06) through active stewardship.

---

## Activities

| Activity | Description |
|----------|-------------|
| Monitoring | Track progress, health, and trajectory against intended outcomes |
| Guidance | Provide direction and input when programs drift or stall |
| Questioning | Challenge assumptions, surface risks, probe for clarity |
| Resourcing | Ensure programs have what they need to succeed |
| Unblocking | Remove obstacles, escalate, make decisions |
| Coordinating | Align across programs, resolve conflicts, manage dependencies |

---

## Stewardship Cadence

| Cadence | Activity |
|---------|----------|
| Weekly | Review progress signals, flag concerns |
| Bi-weekly | Program check-ins (rotating) |
| Monthly | Cross-program coordination review |
| Quarterly | Outcome assessment vs OKRs |

---

## Key Questions

Questions to ask regularly:

- Are we on track to achieve intended outcomes?
- What's blocking progress?
- What decisions are needed?
- Are resources allocated correctly?
- Are dependencies being managed?
- What's changed since last check-in?
- What risks have emerged?

---

## Success Criteria

- [ ] All programs have clear outcome definitions
- [ ] Progress visible and tracked
- [ ] Blockers surfaced and resolved in <1 week
- [ ] Cross-program dependencies managed
- [ ] Quarterly outcomes on track
